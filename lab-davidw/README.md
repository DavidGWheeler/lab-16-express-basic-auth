# lab 16 basic auth
----
## Goal
Create a user model, a POST and a GET route to interact with it, and some tests.

## Setup

 - **make sure you have Node.js and MongoDB installed.**
```$ apt-get node```
```$ npm install mongoose```

 - **install the app**
```$ npm install ```

 - **start the server**
```$ npm run start```

 - **start the database**
```$ mongo```
```$ use userEnv```
